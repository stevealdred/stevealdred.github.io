Sales Analysis Dashboard

The Premium Suites sales department has a continuous ask of suite customer food and beverage spending.

To assist the Food & Beverage Premium Suites department, this dashboard was created to allow the Sales team to view customer spending at any time they required it. It is sourced from the point of sale system with the data queried directly from the SQL server. It allows the user to drill down for specific periods, event types and customers.

As this was created using Excel 2010, advanced VBA was used to prepare and present the data. The company did not support Power Query at the time of build to manual refreshes were required which the user was able to perform.
